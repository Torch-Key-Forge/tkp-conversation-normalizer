# Relationship to Project Foreman

This repository is a reusable upstream component.

```text
Conversation export
        ↓
TKP Conversation Normalizer
        ↓
Decision and Authority Intelligence
        ↓
Conversation-to-Artifact Pipeline
        ↓
Project Foreman
```

Project Foreman is the product-facing recovery experience. This repository is the normalization workshop beneath it and should not be presented as a competing product.
